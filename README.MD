# Node Bases

Tabla de multiplicar realizada con Node.js

```
Opciones:
      --help     Muestra ayuda                                        [booleano]
      --version  Muestra número de versión                            [booleano]
  -b, --base     Tabla de mumeros                           [número] [requerido]
  -l, --listar   Lista en consola la lista           [booleano] [defecto: false]
  -h, --hasta    El límited de cálculo de la tabla        [número] [defecto: 10]
```
